About

Neubloc Timesheet controlling app

------------
External requirements
------------

* python2-gobject
* python2-gconf
* zenity

If running in Ubuntu 11.04
    sudo apt-get install gir1.2-gtk3 libgtk-3.0 zenity

------------
Installation
------------

    sudo pip install -U git+git://github.com/neubloc/neubloc-timesheet.git
